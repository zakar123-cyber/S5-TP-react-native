# My React Native Projects

This repository contains **two separate React Native projects**, each in its own folder. Each project demonstrates a different functionality.

---

## 1. `profile-static`

This project is a **simple greeting app**.

**Features:**
- Input field to enter your name.
- Button to display a greeting message on the screen.
- Demonstrates basic use of `TextInput`, `Button`, and `Text` in React Native.

**How it works:**
1. Enter your name in the text input.
2. Press the "Say Hello" button.
3. Your greeting appears below the button.

---

## 2. `calcul-imc`

This project is a **BMI (IMC) calculator**.

**Features:**
- Input fields for **weight** (kg) and **height** (cm).
- Button to calculate BMI.
- Displays your **BMI value** and a corresponding **health status** with images:
  - Maigreur
  - Normal
  - Surpoids
  - Obésité
  - Obésité sévère
- Demonstrates state management (`useState`) and conditional rendering in React Native.

**How it works:**
1. Enter your weight and height.
2. Press "Calculer IMC".
3. Your BMI and corresponding health image appear on the screen.
